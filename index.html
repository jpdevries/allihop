Allihop

Allihop is a HTML/CSS library designed to help you create clean, accessible, responsive websites.

What will be included

Once complete, the library will include:

  • Code samples and usage notes for dozens of accessible, responsive components
  • A range of colour/font themes tested for WCAG 2.0 Level AA accessibility
  • Optional automated navigation components
  • Subtle but meaningful CSS animations, unambiguously controlled by the user
  • Full responsiveness to font size settings, zoom settings and device sizes
  • All components based on native HTML elements
